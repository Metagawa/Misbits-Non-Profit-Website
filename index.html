Information for Parents

Our Aims and Goals:

  • To provide children with an understanding of modern development practices.
  • To provide them with the tools they need to grow and learn in the tech field.
  • To guide them to a career path in software development.

Our available courses, more coming soon!

These courses run at all times of the year. Simply sign up at any time and your child will be added to the start of the next course! Much of our material draws from the fantastic w3schools.com and we highly recommend checking this out if your child can't attend our courses for any reason.

HTML and CSS: 6 weeks

This course covers all you need to know about making a website just like the one you're viewing right now! The aim of this course is to get the children to make their own website about whatever they want! We tend to get them started by making a wiki page for their favourite tv show or character first and build from there.

Java Programming Summer Course: 8 weeks

An award winning 8 week summer course that covers Java Programming from the very basics. It starts with a quick 2 week refresher course that introduces the children to using computers and the concepts behind coding.

Python Programming Game Dev Summer Course: 10 weeks

This course is run in partnership with renpy.org to teach kids two things. The first is basic coding in Python, the second is how to make a basic game in the renpy engine. The development of the game will be considered the project for this course. It involves breaking the class into groups and having each write their own story and implementing it in each group's game!
