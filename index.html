Information for Parents

Our Aims and Goals:

  • To provide children with an understanding of modern development practices.
  • To provide them with the tools they need to grow and learn in the tech field.
  • To guide them to a career path in software development.

HTML and CSS 6 week course

This course covers all you need to know about making a website just like the one you're viewing right now! The course runs for 6 weeks at all times of the year. Simply Sign up at any point and your child will be added to the start of the next course!

Java Programming Summer Course: ages 12-16

An award winning 8 week summer course that covers Java Programming from the very basics. It starts with a quick 2 week refresher course that introduces the children to using computers and the concepts behind coding.
